# 🛡️ Calculadora de Partidas Rankeadas

Este projeto foi desenvolvido como parte do bootcamp de lógica de programação da **DIO**. O objetivo é aplicar conceitos fundamentais de desenvolvimento de software para classificar o nível de um herói baseado em seu desempenho em partidas.

## 🧠 Conceitos Aplicados
- **Variáveis e Operadores:** Para manipulação de dados e cálculos matemáticos.
- **Estruturas de Decisão (if/else):** Para determinar o ranking do jogador.
- **Funções:** Para organizar e reutilizar a lógica de cálculo.

## 🛠️ Tecnologias
- JavaScript
- Node.js (Ambiente de execução)
- Git & GitHub (Controle de versão)
